
# 概要
このリポジトリは、「仕事術2.0」の note の XML データを 1-記事 1-markdown-file に変換したものである。AI エージェントが仕事術2.0を使えるようにするためだ。

# ディレクトリ構成
- note-workhack20-1.xml
    - note からエクスポートしたオリジナルのデータ
    - バージョン管理してないため、手元に存在しない場合がある。作者が手作業で配置させる必要がある
- index.md
    - 記事ファイルの目次
- articles/
    - 記事ファイルを入れたディレクトリ
- convert.py
    - XML オリジナルデータから記事ファイルをつくるスクリプト
